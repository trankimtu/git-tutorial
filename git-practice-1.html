# git-practice-1 #use for current html version Git-practice-1: Create new repository

I. Create new repository

https://github.com/trankimtu Create a new repository name "git-practice-1.git" Copy clone URL

II. Clone to laptop

a. Open Git Bash b. Use linux command go to folder want to save pwd: current directory ls: list cd: go to folder cat: display file content nano: text editor for programming c. Use command "git clone" to clone everything to laptop Command: git clone https://github.com/trankimtu/git-practice-1.git d. Use command "git remote" to set a name for this URL Command: git remote add origin https://github.com/trankimtu/git-practice-1.git // origin: default name, but can choose any name e. Use command "git remote" to know current remote: (*_*) After clone to laptop, origin is automatical created (*_*)

III. Create content

Use Visual Studio Code to code the html

IV. Post Online

1. Send to staging area - "git add"

git add FullFileName Ex: git add homepage.html or git add --all

2. Send to local repository - git commit

git commit -m " /*write a comment what is the change */ "

3. Move to remote repository

git push origin or git push. or git push
